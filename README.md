## Run Locally

Clone the project

```bash
  git clone https://github.com/smiletondi/AjiCreative-challenge.git
```

Go to the project directory

```bash
  cd AjiCreative-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
